# CitiSync

### CitiSync is designed to empower citizens by providing a platform where they can report issues, collaborate on solutions, and foster community engagement for a safer and more inclusive society.

## Tech Stack Used:

- NextJS
- Tailwind
- Firebase
- Supabase
- GoogleMaps API
